# 图书管理系统 java238

## 项目概述

本系统是基于 Spring Boot 与 Vue.js 构建的图书管理系统，实现前后端分离，具备会员管理、图书列表管理、图书管理和借书管理等主要功能模块。

## 技术栈

### 后端

- Spring Boot
- MyBatis
- Maven
- MySQL 8

### 前端

- Vue.js（Node.js版本14以上，npm版本6.9以上）

## 功能模块

- 会员管理
- 图书列表管理（含图书图片上传功能，存储在本地）
- 图书管理
- 借书管理
- 管理员管理

## 系统角色

- 会员
- 管理员

## 运行环境

- 后端：Java 1.8+
- 前端：现代浏览器
- 数据库：MySQL 8

## 部署步骤

1. 搭建后端运行环境（Java 1.8+，MySQL 8）
2. 部署后端服务（执行Maven构建，部署到服务器）
3. 搭建前端运行环境（Node.js 14+，npm 6.9+）
4. 前端项目构建与部署

## 目录结构

```
书籍管理系统
├── backend    后端代码
├── frontend   前端代码
└── sql        数据库脚本
```

## 核心亮点

- 前后端分离，易于维护与扩展
- 完善的图书管理功能，支持图片上传
- 灵活的借书管理，满足日常图书借阅需求
